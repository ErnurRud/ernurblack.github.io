Ernur Black — Личный сайт

Ernur Black — это личный сайт, созданный как цифровое отражение характера, мыслей и интересов разработчика Ернура. Это не просто страница “о себе”, а интерактивное пространство с уникальной атмосферой, сочетающее в себе тёмный дизайн, анимации и элементы геймификации.

Проект разработан с использованием HTML, CSS и JavaScript без сторонних фреймворков, что позволило полностью контролировать внешний вид и поведение сайта.

🔹 Основные особенности:

* Анимированный фон (частицы, звёзды, снег)
* Кастомный курсор
* Музыкальный плеер с сохранением состояния
* Эффект печатающегося текста
* Интерактивные анимации при скролле
* Мини-игры (кликер и “угадай число”)
* Секретная пасхалка (скрытая комбинация клавиш)
* Анимированный canvas-скелет

🎯 Цель проекта:

Создать уникальный личный сайт с атмосферой, отражающей индивидуальность автора, а также улучшить навыки веб-разработки и работы с JavaScript.

Ernur Black — Personal Website

Ernur Black is a personal website created as a digital reflection of Ernur’s personality, thoughts, and interests. It is not just a simple “about me” page, but an interactive experience with a unique dark aesthetic, animations, and game-like elements.

The project was built using pure HTML, CSS, and JavaScript without external frameworks, allowing full control over design and functionality.

🔹 Key Features:

* Animated background (particles, stars, snow)
* Custom cursor
* Music player with saved state
* Typing text effect
* Scroll-based animations
* Mini-games (clicker & number guessing game)
* Hidden easter egg (secret key combination)
* Animated canvas skeleton

🎯 Project Goal:

To create a unique personal website that reflects the author’s identity and style, while improving web development skills and working with JavaScript.
